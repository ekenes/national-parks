# Mapping the change in annual visits U.S. national parks

The apps included in this repository visualize how annual visits to U.S. national parks has fluctuated over time. The methodology behind each visualization is described in the following blog post: [Visualize diverging datasets with size in web maps](https://www.esri.com/arcgis-blog/products/js-api-arcgis/mapping/visualize-divergâ€¦size-in-web-maps/).
